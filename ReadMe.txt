This is Demostration of using web services

2019410clientapp - Client Side Windows Form App
2019410websrvice - Sever side web service

Database
- Microsoft SQL Sever Database

Guide
- Create the database in sql sever and create table called Student with attribute of StudentId(int), StudentName(varchar(50)), Age(int)
- Create Model1 on web service app using that database table (using .NET Entity Realtion Framework)
- Run the web service
- Create ServiceReference1 in client app using that web service
- Run the Client App
